# Gradle Task :

* Publish to MavenLocal

```
./gradlew clean publishToMavenLocal
```

* Run to build

```
./gradlew build
``` 

* Run in terminal

```
java -jar utils/build/libs/utils-1.3.5.jar
``` 

* Run in terminal

```
java -jar api/build/libs/api-1.3.5.jar
```

* Run greeting plugin

```
./gradlew hello
```
